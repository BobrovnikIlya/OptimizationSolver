Руководство пользователя

При переходе на сайт пользователя встретит начальная страница, где сверху можно наблюдать меню навигации, а ниже условие задачи



На главной странице пользователю предоставляется возможность ввести значения необходимые для решения задачи и выбрать метод решения. После ввода всех необходимых данных необходимо нажать кнопку «Решить». Если при выборе метода решения был выбран симплекс-метод, то пользователь попадет на отдельную страницу «Симплекс-метод» с результатами решения заданной задачи, где будут отображены условие, математическая модель задачи и ответ





При выборе решения методом Хука-Дживса вывод результатов будет аналогичен симплекс-методу, но с присутствием графика содержащем путь от начальной до конечной точки.




Пользователь так же имеет возможность передвигаться по сайту с помощью меню, где присутствуют страницы с решениями методов, «Об авторе» и «Руководство пользователю»




На странице «Главная» имеется возможность отредактировать значения как по средствам простого ввода, так и с помощь кнопок, которые увеличиваю и уменьшают значения полей на определенное количество процентов. После чего провести повторный расчет и сравнить полученные результаты.
